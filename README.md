Описание проекта
Данный проект представляет собой простой симулятор криптовалютной биржи, позволяющий пользователям совершать виртуальные операции купли-продажи активов, отслеживать собственный баланс и изменять свою репутацию путем пожертвований на благотворительность.

Основные возможности программы:
Меню пользователя
Программа имеет простое меню, которое позволяет выбрать одно из действий:

Посмотреть свой баланс: отображение текущего состояния баланса игрока.
Идти на биржу: переход на виртуальную биржу для совершения сделок.
Отдать все деньги на благотворительность: сброс текущих денежных средств и повышение репутации.
Посмотреть свою репутацию: проверка уровня репутации игрока.
Биржа
При выборе опции "Идти на биржу" программа генерирует случайную цену актива, предлагая игроку принять решение о покупке. Если игрок соглашается купить актив, исход сделки определяется случайно: либо игрок получает прибыль равную цене покупки, либо теряет вложенные средства.

Баланс и Репутация
Игрок начинается с фиксированной суммы денег ($500). После каждой успешной операции на бирже сумма увеличивается. Игроки также могут жертвовать свои накопления на благотворительность, что сбрасывает их баланс обратно до начальной суммы и повышает уровень репутации.
